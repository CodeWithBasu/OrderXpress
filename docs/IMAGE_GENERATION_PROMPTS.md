# OrderXpress - Image Generation Prompts for Gemini

This document contains detailed prompts for generating all product images needed for the OrderXpress Point of Sale system using Google Gemini's image generation capabilities.

## üìã Overview

The OrderXpress POS system requires **15 product images** across three categories:

- **Food Items** (5 images)
- **Drinks** (5 images)
- **Desserts** (5 images)

## üé® General Guidelines

**Image Specifications:**

- **Format**: PNG (transparent background preferred)
- **Resolution**: 1024x1024 pixels (square)
- **Style**: Photorealistic, appetizing, professional food photography
- **Lighting**: Bright, natural lighting with soft shadows
- **Background**: Clean white or subtle gradient background
- **Angle**: Slight top-down angle (45-60 degrees) for best presentation
- **Quality**: High-resolution, sharp focus, vibrant colors

## üéØ Tips for Best Results

1. **Use Gemini's Advanced Image Model**: Use the latest Gemini image generation model for best quality
2. **Iterate if Needed**: If the first generation doesn't match expectations, refine the prompt
3. **Consistency**: Try to maintain consistent lighting and style across all images
4. **Resolution**: Always request high resolution for crisp display on modern screens
5. **Background**: White or transparent backgrounds work best for product displays
6. **Realism**: Emphasize photorealistic quality for food items to make them appetizing
